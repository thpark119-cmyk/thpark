# Music In One / Mio Store Submission Overview

## App Name
Music In One

## Short Brand Name
Mio

## App Purpose
Music In One is a focused workspace for musicians to manage lesson logs, student records, repertoire notes, tuning, and rhythm practice tools.

## Main Features
- Lesson logs
- Student management
- Local student photo attachments
- Repertoire / score notes
- IMSLP search shortcut
- Chromatic tuner
- Tone generator
- Metronome
- Multilingual UI
- Google sign-in
- Per-user Firestore sync
- Local fallback storage

## Target Users
- Music students
- Classical musicians
- Instrumentalists
- Vocalists
- Private lesson teachers
- Practice-focused musicians

## Current Release Stage
Pre-release / Store preparation
