# Permissions and Purpose Strings

*Note: These strings are for documentation purposes and for when a Capacitor/native wrapper is created. Currently, they are handled by the browser.*

## Microphone

**iOS (`NSMicrophoneUsageDescription`):**

English:
Music In One uses the microphone to analyze pitch for the tuner feature. Audio is not uploaded to our servers.

Korean:
Music In One은 튜너 기능에서 악기와 목소리의 음정을 분석하기 위해 마이크를 사용합니다. 오디오는 서버로 업로드되지 않습니다.

German:
Music In One verwendet das Mikrofon, um die Tonhöhe für die Stimmgerät-Funktion zu analysieren. Audio wird nicht auf unsere Server hochgeladen.

**Android (`RECORD_AUDIO`):**

English:
Microphone access is used only for real-time pitch analysis in the tuner feature.

Korean:
마이크 접근은 튜너 기능의 실시간 음정 분석에만 사용됩니다.

German:
Der Mikrofonzugriff wird nur für die Echtzeit-Tonhöhenanalyse im Stimmgerät verwendet.
